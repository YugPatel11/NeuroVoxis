## 🎯 Usage Guide

### Background Usage

* Main page → `#082032`
* Sections → `#2C394B`
* Cards → `#334756`

### Text Usage

* Headings → `#FFFFFF`
* Paragraph → `#B0BEC5`
* Disabled/Muted → `#6C7A89`

### Button Usage

* Default → `#00ADB5`
* Hover → `#019CA3`

## ⚡ Tailwind CSS Config

```js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: "#082032",
        secondary: "#2C394B",
        card: "#334756",
        accent: "#00ADB5",
      }
    }
  }
}
```

## ✨ UI Enhancement Instructions (Make it Eye-Catching)

Use these instructions when designing the UI:

* Make the design **modern, premium, and visually striking**
* Use **3D elements** like floating cards, soft shadows, and layered sections
* Add **smooth animations** (hover effects, transitions, micro-interactions)
* Use **depth and layering** (background blur, gradients, shadows)
* Add **gradient highlights** using accent colors
* Use **large bold headings** for strong visual impact
* Keep **clean spacing and alignment**
* Add **interactive buttons** with hover glow effects
* Use **subtle motion** (fade-in, slide-up, scale effects)
* Maintain **dark theme consistency** with glowing accents

### 🧊 Advanced UI Effects (Use Smartly)

* **Glassmorphism**:

  * Use for cards, modals, navbar
  * Apply: `backdrop-blur`, semi-transparent background, thin border
  * Example feel: frosted glass UI

* **Neumorphism**:

  * Use for buttons, input fields, toggles
  * Apply: soft inner & outer shadows
  * Keep subtle (don’t overuse on dark UI)

* **Liquid Glass / Glow UI**:

  * Use for hero section, CTA buttons
  * Add gradient glow using accent color
  * Slight blur + shine effect for premium feel

* **3D Depth Effects**:

  * Floating cards with shadow
  * Parallax scrolling (optional)
  * Scale on hover (like 1.02 or 1.05)
